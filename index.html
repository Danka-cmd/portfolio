Pathfinding Visualizer (C++/SDL)
Algorithms • Performance • Desktop tool

I built an interactive application designed to compare the behavior of three classic pathfinding algorithms: A*, Dijkstra’s algorithm, and Breadth-First Search (BFS) on large grid maps. The app provides a clear visual representation of how each algorithm explores the search space and finds the optimal path, allowing users to see differences in efficiency and strategy in real time. To achieve smooth performance even on large grids, I implemented a custom priority queue that significantly reduces memory allocations and improves execution speed. This optimization makes the visualization highly responsive, letting users experiment with various scenarios without noticeable slowdowns.

  • ~3.4× faster than baseline with arena allocation.
  • Visual step‑through, pause/resume, heatmaps for visited nodes.
  • Cross‑platform builds with CMake and GitHub Actions.
